Overview
🎓
Learning Project
Generative AI with LLMs DeepLearning.ai

Completed as part of the DeepLearning.ai course Generative AI with LLMs, which covers foundational LLM concepts, fine-tuning methods and alignment techniques. This project applied course concepts to a practical dialogue summarization task with a rigorous side-by-side evaluation.

The project explores how to adapt a pre-trained language model for a specific task daily-life dialogue summarization using two distinct fine-tuning strategies. The base model is Flan-T5, Google's instruction-tuned T5 variant, loaded from Hugging Face along with the knkarthick/dialogsum dataset of conversational dialogue-summary pairs.

The key question being investigated is whether Parameter Efficient Fine-Tuning (PEFT) using LoRA adapters can match or outperform traditional full fine-tuning while using a fraction of the trainable parameters and compute. The answer validated through ROUGE scores and qualitative evaluation and turns out to be yes: PEFT with LoRA achieves a 5.98% improvement in ROUGE-1 over the original model, outperforming full fine-tuning across all four ROUGE metrics.

Approaches compared
  • 01 Zero-shot baseline with original Flan-T5
  • 02 Full fine-tuning where all model parameters were updated
  • 03 PEFT/LoRA with only adapter layers trained

Why PEFT / LoRA

Full fine-tuning updates all model parameters which is expensive for large models and risks catastrophic forgetting. LoRA inserts low-rank adapter matrices into attention layers, freezing the original weights and training only the adapters. This reduces trainable parameters by orders of magnitude while preserving the model's general capabilities.

Outcome

The PEFT-enhanced Flan-T5 model demonstrated superior performance over full fine-tuning across all four ROUGE metrics, achieving a 5.98% improvement in ROUGE-1 over the original baseline. This confirms that LoRA-based PEFT is not only more computationally efficient but also more effective for adapting large language models to domain-specific summarization tasks.

The project reinforced a key insight from the LLM fine-tuning literature: updating all model parameters is rarely necessary and can actually hurt performance on narrow tasks through overfitting or catastrophic forgetting. Adapter-based methods like LoRA offer a better balance of efficiency and accuracy, making them the preferred approach for production LLM fine-tuning workflows.
